# PARIS SENTINEL EDITION - GUIDE DE DÉMARRAGE (VERSION ZERO-CONFIG)

Bienvenue sur la version "Paris Sentinel" de World Monitor. 
Cette version a été optimisée pour fonctionner **SANS AUCUNE CLÉ API**. 

L'application utilise désormais des sources OpenData publiques (OpenData Paris et flux directs Sytadin).

## 1. INSTALLATION & LANCEMENT (ULTRA-SIMPLE)

1.  Assurez-vous d'avoir **Node.js (v18+)** et **pnpm** (ou npm) installés.
2.  Dans le terminal, à la racine du projet :
    ```bash
    pnpm install
    pnpm dev
    ```
3.  Ouvrez votre navigateur sur `http://localhost:5173`.

## 2. CONFIGURATION (DÉJÀ PRÊTE)

Le projet est configuré par défaut pour activer le mode Paris. 
Le fichier `.env` n'est plus nécessaire pour les fonctionnalités de base (Trafic, Métro, Caméras).

## 3. FONCTIONNALITÉS ACTIVES (SANS CLÉ)

-   **3D Globe (Next-Gen)** : Une sphère 3D complète avec atmosphère et étoiles.
-   **Trafic Temps Réel** : Données récupérées directement sur le portail OpenData de la Ville de Paris.
-   **Caméras Sytadin** : Flux d'images en direct du Périphérique (Porte de la Chapelle, Maillot, Italie, etc.).
-   **Réseau Métro/RER** : Visualisation complète avec les couleurs officielles des lignes.
-   **UI Tactical 2026** : Interface sombre, effets de verre dépoli (glassmorphism) et animations cyan.

## 4. DÉPANNAGE

-   **Carte Noire** : Votre navigateur doit supporter WebGL2.
-   **Pas de Données** : Vérifiez que vous avez bien une connexion internet active pour charger les flux distants.
-   **Zoom** : Le focus Paris s'active automatiquement, mais vous pouvez naviguer sur tout le globe.
